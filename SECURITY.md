# Security Policy

## Reporting a Vulnerability

Report security issues privately via [GitHub's security advisory feature](https://github.com/itsOmidKarami/jellygrab/security/advisories/new) or email itsomidkarami@gmail.com.

Please include:
- Description of the issue
- Steps to reproduce
- Potential impact

Expect a response within 7 days. Do not open a public issue for security vulnerabilities.

## Scope

JellyGrab is designed to run on a **private Docker network** with no external exposure. There is intentionally no authentication. Running the sidecar on a public interface is outside the supported deployment model.
